# Generics Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add generics (type parameterization) to Bengal with Swift-like `<T: Bound>` syntax, supporting generic functions and structs with single protocol constraints and monomorphization.

**Architecture:** The implementation extends the existing pipeline (lexer -> parser -> semantic -> BIR -> LLVM) without changing BIR or codegen. A new monomorphization pass between semantic analysis and BIR lowering substitutes type parameters with concrete types. The `<` vs comparison ambiguity is resolved by a new language rule requiring spaces around infix operators.

**Tech Stack:** Rust, logos (lexer), inkwell (LLVM bindings)

**Spec:** `docs/superpowers/specs/2026-03-25-generics-design.md`

fn parse_type_params(&mut self) -> Result<Vec<TypeParam>> {
    let mut params = Vec::new();
    self.expect(Token::Lt)?;
    loop {
        let name = self.expect_ident()?;
        let bound = if self.peek().node == Token::Colon {
            self.advance();
            Some(self.expect_ident()?)
        } else {
            None
        };
        params.push(TypeParam { name, bound });
        if self.peek().node == Token::Comma {
            self.advance();
        } else {
            break;
        }
    }
    self.expect(Token::Gt)?;
    Ok(params)
}

fn parse_type(&mut self) -> Result<TypeAnnotation> {
    if self.peek().node == Token::LParen {
        self.advance();
        self.expect(Token::RParen)?;
        return Ok(TypeAnnotation::Unit);
    }
    let tok = self.expect(Token::Ident(String::new()))?;
    let base = match &tok.node {
        Token::Ident(s) if s == "Int32" => return Ok(TypeAnnotation::I32),
        Token::Ident(s) if s == "Int64" => return Ok(TypeAnnotation::I64),
        Token::Ident(s) if s == "Float32" => return Ok(TypeAnnotation::F32),
        Token::Ident(s) if s == "Float64" => return Ok(TypeAnnotation::F64),
        Token::Ident(s) if s == "Bool" => return Ok(TypeAnnotation::Bool),
        Token::Ident(s) if s == "Void" => return Ok(TypeAnnotation::Unit),
        Token::Ident(s) => s.clone(),
        _ => unreachable!(),
    };
    // Check for type arguments: `<` immediately after ident (no space)
    if self.peek().node == Token::Lt && self.no_space_before_current() {
        self.advance(); // consume `<`
        let mut args = vec![self.parse_type()?];
        while self.peek().node == Token::Comma {
            self.advance();
            args.push(self.parse_type()?);
        }
        self.expect(Token::Gt)?;
        Ok(TypeAnnotation::Generic { name: base, args })
    } else {
        Ok(TypeAnnotation::Named(base))
    }
}

/// Returns true if there is no whitespace between the previous token and the current token.
fn no_space_before_current(&self) -> bool {
    if self.pos == 0 {
        return false;
    }
    let prev = &self.tokens[self.pos - 1];
    let curr = &self.tokens[self.pos];
    prev.span.end == curr.span.start
}
```

fn parse_postfix(&mut self) -> Result<Expr> {
    let mut expr = self.parse_primary()?;
    loop {
        match &self.peek().node {
            Token::Dot => {
                // ... existing dot handling unchanged ...
            }
            Token::Lt if self.no_space_before_current() => {
                // Type arguments: ident<Type, ...>(args)
                let name = match &expr.kind {
                    ExprKind::Ident(name) => name.clone(),
                    _ => break,
                };
                let type_args = self.parse_type_arg_list()?;
                // Must be followed by `(`
                if self.peek().node != Token::LParen {
                    return Err(BengalError::ParseError {
                        message: "expected `(` after type arguments".to_string(),
                        span: self.peek().span,
                    });
                }
                expr = self.parse_postfix_call_with_type_args(name, type_args)?;
            }
            Token::LParen => {
                expr = self.parse_postfix_call(expr)?;
            }
            _ => break,
        }
    }
    Ok(expr)
}
```

- [ ] **Step 6: Add parse_type_arg_list and parse_postfix_call_with_type_args helpers**

```rust
fn parse_type_arg_list(&mut self) -> Result<Vec<TypeAnnotation>> {
    self.expect(Token::Lt)?;
    let mut args = vec![self.parse_type()?];
    while self.peek().node == Token::Comma {
        self.advance();
        args.push(self.parse_type()?);
    }
    self.expect(Token::Gt)?;
    Ok(args)
}

fn parse_postfix_call_with_type_args(
    &mut self,
    name: String,
    type_args: Vec<TypeAnnotation>,
) -> Result<Expr> {
    self.expect(Token::LParen)?;

    if self.peek().node == Token::RParen {
        self.advance();
        return Ok(self.expr(ExprKind::Call {
            name,
            type_args,
            args: vec![],
        }));
    }

    // Lookahead: IDENT followed by `:` → named args → StructInit
    let is_named = matches!(self.peek().node, Token::Ident(_))
        && self.tokens.get(self.pos + 1).map(|t| &t.node) == Some(&Token::Colon);

    if is_named {
        let mut args = Vec::new();
        let label = self.expect_ident()?;
        self.expect(Token::Colon)?;
        let value = self.parse_expr()?;
        args.push((label, value));
        while self.peek().node == Token::Comma {
            self.advance();
            let label = self.expect_ident()?;
            self.expect(Token::Colon)?;
            let value = self.parse_expr()?;
            args.push((label, value));
        }
        self.expect(Token::RParen)?;
        Ok(self.expr(ExprKind::StructInit {
            name,
            type_args,
            args,
        }))
    } else {
        let mut args = Vec::new();
        args.push(self.parse_expr()?);
        while self.peek().node == Token::Comma {
            self.advance();
            args.push(self.parse_expr()?);
        }
        self.expect(Token::RParen)?;
        Ok(self.expr(ExprKind::Call {
            name,
            type_args,
            args,
        }))
    }
}
```

fn resolve_type_with_type_params(

    type_params: &[TypeParam],
    resolver: &Resolver,

    match annotation {
        TypeAnnotation::Named(name) => {
            // Check if it's a type parameter
            if let Some(tp) = type_params.iter().find(|tp| tp.name == *name) {
                Ok(Type::TypeParam {
                    name: tp.name.clone(),
                    bound: tp.bound.clone(),

                .iter()

                .collect::<Result<Vec<_>>>()?;
            Ok(Type::Generic {
                name: name.clone(),
                args: resolved_args,
            })
        }

    }

fn substitute_type(ty: &Type, type_params: &[TypeParam], concrete: &[Type]) -> Type {
    match ty {
        Type::TypeParam { name, .. } => {
            for (tp, concrete_ty) in type_params.iter().zip(concrete) {
                if tp.name == *name {
                    return concrete_ty.clone();
                }
            }
            ty.clone()
        }
        Type::Generic { name, args } => Type::Generic {
            name: name.clone(),
            args: args.iter().map(|a| substitute_type(a, type_params, concrete)).collect(),
        },
        _ => ty.clone(),
    }
}

fn substitute_types(types: &[Type], type_params: &[TypeParam], concrete: &[Type]) -> Vec<Type> {
    types.iter().map(|t| substitute_type(t, type_params, concrete)).collect()

fn check_protocol_conformance(ty: &Type, protocol_name: &str, resolver: &Resolver) -> Result<()> {
    let struct_name = match ty {
        Type::Struct(name) => name,
        _ => return Err(sem_err(format!(
            "type '{}' does not conform to protocol '{}'", ty, protocol_name
        ))),
    };
    // Check that the struct conforms to the protocol
    // Look up struct conformances from the AST or a conformance map
    // For now, check if the struct's methods satisfy the protocol's requirements
    let proto = resolver.lookup_protocol(protocol_name)
        .ok_or_else(|| sem_err(format!("unknown protocol '{}'", protocol_name)))?;
    let struct_info = resolver.lookup_struct(struct_name)
        .ok_or_else(|| sem_err(format!("unknown struct '{}'", struct_name)))?;

    for req_method in &proto.methods {
        if !struct_info.method_index.contains_key(&req_method.name) {
            return Err(sem_err(format!(
                "type '{}' does not conform to protocol '{}': missing method '{}'",
                struct_name, protocol_name, req_method.name
            )));
        }
    }
    for req_prop in &proto.properties {
        if !struct_info.field_index.contains_key(&req_prop.name)
            && !struct_info.computed_index.contains_key(&req_prop.name)
        {
            return Err(sem_err(format!(
                "type '{}' does not conform to protocol '{}': missing property '{}'",
                struct_name, protocol_name, req_prop.name
            )));
        }
    }
    Ok(())
}

```rust
use std::collections::{HashMap, HashSet};

use crate::parser::ast::*;
use crate::semantic::types::Type;

/// A concrete instantiation: (generic_name, concrete_type_args)
#[derive(Debug, Clone, PartialEq, Eq, Hash)]
pub struct Instantiation {
    pub name: String,
    pub type_args: Vec<TypeAnnotation>,
}

/// Monomorphize a program: replace all generic definitions and calls with
/// specialized (concrete) versions.
pub fn monomorphize(program: &Program) -> Program {
    let mut collector = InstantiationCollector::new(program);
    collector.collect();

    let mut output = program.clone();

    // Remove generic functions and structs from the output
    output.functions.retain(|f| f.type_params.is_empty());
    output.structs.retain(|s| s.type_params.is_empty());

    // Generate specialized versions
    for inst in &collector.func_instantiations {
        if let Some(generic_func) = program.functions.iter().find(|f| f.name == inst.name) {
            let specialized = specialize_function(generic_func, &inst.type_args);
            output.functions.push(specialized);
        }
    }
    for inst in &collector.struct_instantiations {
        if let Some(generic_struct) = program.structs.iter().find(|s| s.name == inst.name) {
            let specialized = specialize_struct(generic_struct, &inst.type_args);
            output.structs.push(specialized);
        }
    }

    // Rewrite all call sites and struct inits to use specialized names
    rewrite_calls(&mut output, &collector);

    output
}
```

This is the most complex task. The full implementation includes:
- `InstantiationCollector`: walks all expressions to find `Call`/`StructInit` with non-empty `type_args`
- `specialize_function`: substitutes type params in a function definition, renames to `name_Type1_Type2`
- `specialize_struct`: substitutes type params in a struct definition and its methods
- `rewrite_calls`: rewrites all `Call`/`StructInit` to use the specialized name and clear `type_args`
- `mangle_instantiation`: produces the specialized name (e.g., `identity_Int32`)
